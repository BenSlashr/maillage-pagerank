Notre crawler analyse automatiquement votre site web pour extraire :

  • Le contenu de vos pages (titres, textes)
  • La structure des liens internes existants
  • Les métadonnées importantes (balises title, h1, etc.)

Il détecte automatiquement les segments de votre site (catégories, produits, articles, etc.) pour faciliter la configuration des règles de maillage.

Il génère ensuite les fichiers nécessaires à l'analyse de maillage interne, sans que vous ayez à les préparer manuellement.

L'intégration GSC permet d'enrichir l'analyse avec les requêtes réelles des utilisateurs :

  1. Connectez-vous à votre compte Google via notre interface
  2. Sélectionnez le site à analyser et la période
  3. Importez les données directement dans l'outil d'analyse

Ces données permettent de générer des suggestions d'ancres pertinentes basées sur les requêtes qui amènent du trafic à vos pages.

Connecter GSC

Les règles de maillage par segments sont essentielles pour une stratégie SEO efficace :

  1. Identifiez vos segments : catégories, produits, articles de blog, pages institutionnelles, etc.
  2. Définissez les relations : quels segments peuvent pointer vers quels autres segments
  3. Paramétrez le nombre de liens : combien de liens maximum entre chaque segment

Exemples de règles stratégiques :

  • Articles de blog → Pages produits (pour convertir les lecteurs en clients)
  • Pages catégories → Articles de blog (pour enrichir le contenu des catégories)
  • Pages produits → Produits complémentaires (pour augmenter le panier moyen)
